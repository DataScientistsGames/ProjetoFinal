Descrição detalhada

Classe que representa um tabuleiro de Ligue 4.

Essa classe herda as funções da classe genérica Tabuleiro e especializa a avaliação da condição de finalização do tabuleiro, posicionamento das peças e a impreção da estética do tabuleiro.

◆ finalizarJogo()

int Lig4::finalizarJogo ( )
finaloverridevirtual

Avalia se o tabuleiro atingiu um estado de finalização.

O tabuleiro do Ligue 4, finaliza quando 4 peças de um jogador formam uma linha, coluna ou diagonal.

Retorna
Retorna 0 se o tabuleiro pode continuar a ser usado, retorna 1|2 para o número do jogador vencedor ou retorna 3 para o empate.
